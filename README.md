# 🚀 Zorvyn - Finance Backend System

## 📌 Overview
Zorvyn is a backend system designed to manage financial records with secure role-based access control. It supports user authentication, financial data management, and dashboard analytics for better decision-making.

---

## ✨ Features
- 🔐 JWT Authentication (Login & Register)
- 👥 Role-Based Access Control (Viewer, Analyst, Admin)
- 💰 Financial Records Management (CRUD operations)
- 🔍 Filtering (type, category, date)
- 📄 Pagination support
- 📊 Dashboard Analytics:
  - Total Income
  - Total Expenses
  - Net Balance
  - Category-wise Summary
  - Monthly Trends

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT (Authentication)
- bcrypt.js (Password hashing)
